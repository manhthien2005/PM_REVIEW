▶ RESET PASSWORD (POST /api/v1/auth/reset-password)
  ✅     Token hợp lệ + newPassword ≥ 8 ký tự       → 200 Đặt lại thành công
  ❌     Token hết hạn (>15 phút)                    → 400 Link đã hết hạn
  ❌     Token không hợp lệ                          → 400 Link không hợp lệ
  ❌     Mật khẩu mới = mật khẩu cũ                  → 400 Không được trùng
  ❌     newPassword ≠ confirmPassword                → 400 Không khớp
  ❌     newPassword < 8 ký tự                        → 400 Quá ngắn

▶ CHANGE PASSWORD (PUT /api/v1/auth/password) — cần đăng nhập
