# 🔍 Review Report Template

> Sử dụng template này khi chạy **mode REVIEW**.
> ⚠️ Output PHẢI viết bằng **tiếng Việt** vì để người dùng cuối đọc.

```markdown
# 🔍 Báo Cáo Đánh Giá: [Tên Màn Hình]

## Điểm tổng: X/10
## Confidence: X%

## Checklist đối chiếu Plan
| Mục           | Plan yêu cầu     | Thực tế | Kết quả |
| ------------- | ---------------- | ------- | ------- |
| Loading State | Skeleton shimmer | Có      | ✅ PASS  |
| Error State   | Retry button     | Thiếu   | ❌ FAIL  |

## Lỗi phát hiện
1. 🔴 [Nghiêm trọng] ...
2. 🟡 [Trung bình] ...
3. 🟢 [Nhẹ] ...

## Đề xuất sửa
- ...
```
